# Home Defense Website

A modern Node.js + JavaScript home defense control system using React JSX components on the frontend.

## Features

- Arm / Disarm controls
- Real-time status display
- History page for arm/disarm events
- Express backend API
- React frontend with JSX components
- Shared CSS styling
- No standalone HTML files required in the frontend source

## Run locally

1. Install dependencies:
   - npm install
2. Start development server:
   - npm run dev
3. Open:
   - http://localhost:3000
